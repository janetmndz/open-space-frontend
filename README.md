# Open Space (Frontend)

## Description
A React frontend for Mod5 final project at Flatiron School.

## Dependecies
- create-react-app
- react-router-dom
- node-sass

# MVP Goals
- [x] Index of all posts to a user
- [ ] Show page for each post with comments
- [x] Subscribe ability for topics
- [ ] Ability to add comments to posts
- [ ] CRUD for posts
- [x] Auth
- [x] Nav Bar
- [x] Custom CSS
- [ ] Filter for topics
